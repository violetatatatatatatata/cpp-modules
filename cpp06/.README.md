### ex00
Se pasa como argumento un número y se ha de castear a char, int, float y double. Si hay un error, enseña un mensaje de error.

- La función detectType compara el string del literal con casos especiales predefinidos ("nan", "nanf", "+inf", "-inf", "+inff", "-inff"). Si hay coincidencia, retorna PSEUDO.
- Verifica si es un char, con una longitud exactamente de 3 (incluyendo las comillas)
- Para comprobar decimales, primero busca el '.' y luego mira si hay errores y si la cadena acaba en f es un float.
- Si no cumple ninguno, entonces es un int.

Los **pseudoliterales** son strings que son imposibles de ser representados de forma numérica.

####1. Valores de punto flotante (float)
> Se identifican por llevar una f al final, siguiendo la convención de C++ para literales de precisión simple:
 <br>-inff: Infinito negativo.
 <br>+inff: Infinito positivo.
 <br>nanf: "Not a Number" (No es un número), utilizado para resultados de operaciones matemáticas indefinidas, como dividir cero entre cero.

#### 2. Valores de doble precisión (double)
> Representan los mismos conceptos pero para el tipo double, por lo que no incluyen el sufijo f:

 <br>-inf o inf: Infinito negativo o positivo.
 <br>+inf: Infinito positivo explícito.
 <br>nan: "Not a Number" de doble precisión.
