## CPP 00
---
by: Violeta Velandria

Este módulo introduce los conceptos básicos de C++, como la entrada/salida estándar (streams), clases básicas y la gestión de memoria estática.

### ex00
Implementa el programa 'megaphone'. Recibe parámetros por línea de comandos y los imprime en mayúsculas usando `toupper`. Si no recibe argumentos, muestra el mensaje: `* LOUD AND UNBEARABLE FEEDBACK NOISE *`.

### ex01
Se implementa una agenda telefónica ('PhoneBook') que puede almacenar un máximo de 8 contactos ('Contact'). 
Si se intenta añadir un 9º contacto, el más antiguo es reemplazado por el nuevo. No se permite el uso de asignación dinámica de memoria.

### ex02
Recreación de un diario de transacciones bancarias utilizando la clase 'Account'. El ejercicio se centra en el uso de miembros y funciones estáticas para llevar un registro global de depósitos, retiros y estados de cuenta.
