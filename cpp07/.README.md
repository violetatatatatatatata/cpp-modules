### ex00
Los templates son un mecanismo que permite escribir código genérico que recibe diferentes tipos de datos sin tener que reescribir código.
Estos no actúan sobre el código, sólo durante la compilación. El template muere junto a la función, por eso pueden llamarse igual diferentes _template <typename T>_
